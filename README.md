## ğŸš€ Getting Started

### Prerequisites

- Node.js (LTS version)
- Yarn
- Expo CLI
- iOS Simulator (for Mac) or Android Studio (for Android development)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/rn-skilllab.git
   cd rn-skilllab
   ```

2. Install dependencies:

   ```bash
   yarn install
   ```

3. Start the development server:
   ```bash
   yarn dev
   ```
